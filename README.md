What it does
A healthcare management system demo with domain models (Patient, Doctor, Appointment, Prescription, MedicalRecord) and a TestNG test suite.
Main purpose: Demonstrate TestNG testing practices:
5 test classes with 56+ tests
2 test suites (Unit and Integration)
All Before/After method types
SoftAssert for assertions
TestNG listener for execution tracking
Tests run via Maven Surefire plugin from the terminal
Domain functionality: Models healthcare entities (patients, doctors, appointments, prescriptions, medical records) with static inner classes, enums, and validation logic.
Testing focus: Shows how to structure, organize, and execute TestNG tests using Maven Surefire, including test suites, lifecycle hooks, and listeners.
